# Inspiration / reference images

Drop reference art here for the icon overhaul. Claude studies these to match the
visual style (color, line weight, lighting, level of detail) when hand-crafting
the SVG weapon + landing icons.

These files are **reference only** — they are not shipped or imported by the app.

## How to use
1. Add image files here (`.png`, `.jpg`, `.webp` — any size).
2. Name them descriptively so the intent is clear, e.g.:
   - `dagger.png`, `longbow.png`, `crossbow.png`, `staff.png`, `sling.png`
   - or a single `weapons-sheet.png` / `style-reference.png` for overall vibe
   - `landing-style.png` for the marketing-page icon direction
3. Tell Claude when they're in — it'll review them and build matching SVGs.

(If a name maps to a specific weapon in the picker it helps, but a general
style sheet is fine too — the goal is to capture the *look*.)
